Definition 5.18.1.

A sum family is a object \(S\) together with maps \(C_i \xrightarrow{s_i} S\) (one for each \(i\)), having the universal property:
Given any object \(Y\) and any maps \(C_i \xrightarrow{g_i} Y\text{,}\) there is exactly one map \(S \xrightarrow{g} Y\) such that all the triangles below commute, i.e. such that \(g s_i = g_i\) for each \(i\) in \(I\text{.}\)
